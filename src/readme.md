# Prototype Panic

Prototype Panic är ett spel byggt med Java och JavaFX.

---

## Installation

1. Ladda ner eller klona projektet som en `.zip`-fil
2. Packa upp zip-filen
3. Öppna projektmappen i en IDE, till exempel:
    - IntelliJ IDEA
    - Visual Studio Code

---

## Köra spelet

Projektet använder **Maven** och **JavaFX**.

### Steg:

1. Öppna projektet i din IDE
2. Gå till **Maven-panelen** (vanligtvis till höger)
3. Navigera till:

Plugins → javafx

4. Kör:

javafx


---

## Byggt med

- Java
- JavaFX
- Maven

---

## Om spelet

Prototype Panic är ett spel där spelaren är i mitten och bemöts av monster som attackerar en, använd vapen och uppgradering för att anpassa din karaktär till din egen prototyp.
